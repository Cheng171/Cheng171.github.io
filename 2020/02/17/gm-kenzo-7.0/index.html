发表于 2020-02-17 更新于 2020-02-24

GearMagic - Cheng@HMNote3Pro_Qcom_DEV

20200224 - V10.2.1.0 - MIUI10 - Reburn Release V2.2

VIEW

0-全局高仿 MIUI 11，带给你全新的视觉体验
1-基于Android 7.0 移植自小米Max Pro
2-全局Zipalign（增强软件启动速度）
3-高级电源菜单，支持重启到Rec&Fastboot
4-集成Magisk v19.3
5-纯净无推广软件
6-秒截图
7-状态栏天气
8-状态栏网速秒刷新
9-GPU能耗优化
10-破解卡米验证，随意删除系统软件不卡米
11-新增实用功能
—-WIFI密码查看（设置-WLAN）
—-双击唤醒临时＆永久开启（设置-显示）
—-LCD分辨率&屏幕密度调节（设置-显示）
—-调度一键切换，集成省电均衡游戏共五个调度可供任意切换（设置-特色功能）
—-系统&用户应用冻结（设置-特色功能）
—-电量伪装 手机外借不尴尬（设置-特色功能）
….

TIPS

1-默认使用X27内核，续航和流畅度都还不错，目的是使系统更稳定
（PS:不建议自行换内核，除非为了更好的游戏体验）
2-本包只在酷安&GM官网&GM用户群&kenzo基安py群＆个人博客发布，如果您从其他渠道获得本包，可能会含有推广软件。

BUGS

1-FPC指纹传感器无法进行指纹支付（证书问题，第三方MIUI通病，可以用模块解决）
2-系统自带录屏软件的声音无法设置系统内录（可用第三方录屏软件替代）
3-Goodix汇指纹传感器不能够使用（刷完发现没有指纹录入选项的说明你的手机是Goodix指纹传感器）
